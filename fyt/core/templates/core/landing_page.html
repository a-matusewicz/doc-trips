Trips Numbering

The following numbering convention was adopted in 2005 to make the trip numbers more consistent.

Hundreds Digit (Nxx)

The hundreds digit is the type of trip (blank for zero/Hiking):

  • xx = Hiking
  • 1xx = Paddling
  • 2xx = Climbing
  • 3xx = Biking
  • 4xx = Fishing
  • 5xx = Horseback Riding
  • 7xx = Nature Appreciation
  • 9xx = Special

Tens Digit (xNx)

The tens digit is the distance from Hanover (roughly — you can ignore this if you need to):

  • x0x = Hanover
  • x1x-x3x = Local (southwest of Whites)
    • for trips based at a cabin: x2x = Hinman Cabin
  • x4x-x6x = Whites
    • for trips based at a cabin: x4x = John Rand Cabin, x5x = Ritchie Smith Cabin, x6x = Billings Lodge
  • x7x-x9x = Second College Grant
    • for trips based at a cabin: x7x = Peaks Cabin, x8x = Alder Brook Cabin, x9x = Stoddard Cabin

Ones Digit (xxN)

The ones digit is usually difficulty, but more generally it breaks the type of trip into more categories. In this sense it might make more sense to put this digit in the tens spot, next to the type of trip digit, but since it’s usually difficulty, putting it in the ones spot down-plays the difficulty levels of the trip.

For Hiking trips (and roughly for other types of trips), difficulty:

  • xx0-xx1 = Leisurely — no effort neccessary
  • xx2-xx3 = Easy — minor effort neccessary
  • xx4-xx5 = Moderate — some effort neccessary
  • xx6-xx7 = Strenuous — more effort neccessary
  • xx8-xx9 = Advanced — most effort neccessary

For Nature Appreciation trips, type:

  • xx0 = Writing
  • xx1 = Painting
  • xx2 = Photography
